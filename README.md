## 这是什么？

> 让旅行者随时与角色聊天

本项目使用 [女娲.skill](https://github.com/alchaincyf/nuwa-skill) 对《原神》游戏内的角色进行思维蒸馏

## 安装/使用说明

克隆项目到本地后选择你喜欢的角色的技能文件夹，复制到你使用的智能体工具的skills文件夹下

例如
```
genshin.skill/nicole-perspective/ -> .agents/nicole-perspective/
```

或者让 AI 帮你安装

```
帮我安装 https://github.com/DGP-Studio/Genshin.Skill 下的 尼可 技能
```
